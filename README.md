# 高速公路車流資料爬取與分析專案

---

## 專案簡介
本專案致力於自動從交通部高速公路局網站爬取歷年車流資料，並進行資料清理與分析，協助交通趨勢研究與預測。

## 使用說明

1. 請安裝 Python 3.8+  
2. 安裝套件：
    ```bash
    pip install requests beautifulsoup4
    ```
3. 執行程式：
    ```bash
    python main.py
    ```
4. 資料將存放於專案資料夾下 `Web_Crawler/2024/zip_file/`
